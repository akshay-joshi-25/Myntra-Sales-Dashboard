# 📊 **Myntra Sales Dashboard — Summary**
This dashboard provides a comprehensive view of Myntra's sales performance across products, brands, categories, geography, and time. It is built in Power BI and covers data spanning 2021, 2022, and the first quarter of 2023.

# 📌 **Overview**
This PowerBI dashboard contains three different sections:
- High-level metrics
- Vizualizations
- Data Table

# 📂 **Datasets**
The datasets used for this project were publicly made available on Kaggle.
The Myntra Sales dataset contained the following CSV files:
- Customers
- Orders
- Products

These datasets contained data for the period between January 1, 2021 and March 31, 2023.
You can access these files by clicking [here](https://github.com/akshay-joshi-25/Myntra-Sales-Dashboard/blob/main/Customers.xlsx).

# 📈 **High Level Metrics** 
- Total Sales
- Number of Orders
- Unique Customers
- Average Discount (%)
- Average Rating

# 📊 **Vizualizations** 
#### 1. Sales & Discount Trend (Line + Bar Chart) ####
Tracks Sum of Selling Price (bar) and Average Discount % (line) by Year and Month across 2021–2022.
Monthly sales consistently hover around ₹50K–₹100K, with relatively stable discount rates throughout the period.

#### 2. Geographic Distribution (Map) ####
Plots Order Count by City and State across India.
Orders are spread across multiple states including Bihar, Karnataka, Madhya Pradesh, Rajasthan, Punjab, Telangana, and Gujarat, among others.

#### 3. Sales by Category (Donut Chart) ####
Breaks total selling price into four categories:
🔵 Men
🟣 Women
🟠 Kids
🟤 Beauty
Men and Women appear to be the dominant categories by revenue.
